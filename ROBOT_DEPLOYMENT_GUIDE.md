#### ‡∏Ç‡∏±‡πâ‡∏ô‡∏ï‡∏≠‡∏ô‡∏ó‡∏µ‡πà 3: ‡∏™‡∏£‡πâ‡∏≤‡∏á Transformation Matrix (‡πÅ‡∏ö‡∏ö‡∏á‡πà‡∏≤‡∏¢)

**‡∏ß‡∏¥‡∏ò‡∏µ‡∏ó‡∏µ‡πà‡∏á‡πà‡∏≤‡∏¢‡∏ó‡∏µ‡πà‡∏™‡∏∏‡∏î: 4-Point Method**

1. **‡∏ß‡∏≤‡∏á marker 4 ‡∏à‡∏∏‡∏î** ‡∏ö‡∏ô workspace:
   ```
   A (‡∏°‡∏∏‡∏°‡∏ã‡πâ‡∏≤‡∏¢‡∏ö‡∏ô)      B (‡∏°‡∏∏‡∏°‡∏Ç‡∏ß‡∏≤‡∏ö‡∏ô)
   
   
   C (‡∏°‡∏∏‡∏°‡∏ã‡πâ‡∏≤‡∏¢‡∏•‡πà‡∏≤‡∏á)    D (‡∏°‡∏∏‡∏°‡∏Ç‡∏ß‡∏≤‡∏•‡πà‡∏≤‡∏á)
   ```

2. **‡∏ö‡∏±‡∏ô‡∏ó‡∏∂‡∏Å pixel coordinates**:
   ```python
   # ‡∏ñ‡πà‡∏≤‡∏¢‡∏†‡∏≤‡∏û‡πÅ‡∏•‡∏∞‡∏Ñ‡∏•‡∏¥‡∏Å‡∏´‡∏≤ coordinates
   cap = cv2.VideoCapture(0)
   ret, frame = cap.read()
   
   # ‡πÉ‡∏ä‡πâ mouse callback ‡∏´‡∏£‡∏∑‡∏≠ manual
   pixel_points = [
       (100, 50),   # A
       (540, 50),   # B
       (100, 430),  # C
       (540, 430)   # D
   ]
   ```

3. **‡∏ß‡∏±‡∏î robot coordinates ‡∏à‡∏£‡∏¥‡∏á** (‡πÉ‡∏ä‡πâ Dobot Studio):
   ```python
   robot_points = [
       (200, 200),    # A (X, Y in mm)
       (400, 200),    # B
       (200, -200),   # C
       (400, -200)    # D
   ]
   ```

4. **‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì transformation**:
   ```python
   import numpy as np
   
   # Homography matrix
   pixel_pts = np.array(pixel_points, dtype=np.float32)
   robot_pts = np.array(robot_points, dtype=np.float32)
   
   H, status = cv2.findHomography(pixel_pts, robot_pts)
   
   print("Homography Matrix:")
   print(H)
   ```

5. **‡πÉ‡∏ä‡πâ‡∏á‡∏≤‡∏ô**:
   ```python
   def pixel_to_robot(pixel_x, pixel_y, H):
       """‡πÅ‡∏õ‡∏•‡∏á pixel ‚Üí robot coordinates"""
       pixel_point = np.array([[pixel_x, pixel_y]], dtype=np.float32)
   ```

```python
"""
Robot Control for Dobot MG400
"""

import numpy as np
from pydobot import Dobot
import time


class DobotController:
    """Simple Dobot MG400 controller"""
    
    def __init__(self, config, homography_matrix):
        self.config = config
        self.H = np.array(homography_matrix)
        self.robot = None
        self.is_connected = False
    
    def connect(self, port_or_ip):
        """Connect to robot"""
        try:
            self.robot = Dobot(port=port_or_ip)
            self.is_connected = True
            print("‚úì Robot connected")
            return True
        except Exception as e:
            print(f"‚úó Connection failed: {e}")
            return False
    
    def home(self):
        """Home robot"""
        if not self.is_connected:
            print("Not connected")
            return False
        
        self.robot.home()
        print("‚úì Robot homed")
        return True
    
    def pixel_to_robot(self, pixel_x, pixel_y):
        """Convert pixel to robot coordinates"""
        pixel_point = np.array([[pixel_x, pixel_y]], dtype=np.float32)
        pixel_point = pixel_point.reshape(-1, 1, 2)
        
        robot_point = cv2.perspectiveTransform(pixel_point, self.H)
        
        robot_x = float(robot_point[0][0][0])
        robot_y = float(robot_point[0][0][1])
        
        # Validate workspace limits
        robot_x = np.clip(robot_x, self.config.ROBOT_X_MIN, self.config.ROBOT_X_MAX)
        robot_y = np.clip(robot_y, self.config.ROBOT_Y_MIN, self.config.ROBOT_Y_MAX)
        
        return robot_x, robot_y
    
    def execute_grasp(self, grasp, depth_value):
        """Execute a grasp"""
        if not self.is_connected:
            print("Not connected")
            return False
        
        # Convert grasp center to robot coords
        cy, cx = grasp.center
        robot_x, robot_y = self.pixel_to_robot(cx, cy)
        
        # Calculate Z based on depth
        # (‡∏ï‡πâ‡∏≠‡∏á‡∏õ‡∏£‡∏±‡∏ö‡∏ï‡∏≤‡∏°‡∏Å‡∏≤‡∏£ calibrate)
        robot_z = 50  # mm above surface
        
        # Calculate rotation from grasp angle
        robot_r = np.degrees(grasp.angle)
        
        print(f"\nExecuting grasp:")
        print(f"  Pixel: ({cx:.0f}, {cy:.0f})")
        print(f"  Robot: ({robot_x:.1f}, {robot_y:.1f}, {robot_z:.1f})")
        print(f"  Angle: {robot_r:.1f}¬∞")
        print(f"  Quality: {grasp.quality:.3f}")
        
        try:
            # 1. Move to safe height
            self.robot.move_to(robot_x, robot_y, 100, robot_r)
            time.sleep(1)
            
            # 2. Open gripper
            self.robot.suck(False)  # ‡∏´‡∏£‡∏∑‡∏≠‡πÉ‡∏ä‡πâ gripper command
            time.sleep(0.5)
            
            # 3. Move down to grasp
            self.robot.move_to(robot_x, robot_y, robot_z, robot_r)
            time.sleep(1)
            
            # 4. Close gripper
            self.robot.suck(True)  # ‡∏´‡∏£‡∏∑‡∏≠‡πÉ‡∏ä‡πâ gripper command
            time.sleep(1)
            
            # 5. Lift up
            self.robot.move_to(robot_x, robot_y, 100, robot_r)
            time.sleep(1)
            
            # 6. Move to drop position (‡∏Å‡∏≥‡∏´‡∏ô‡∏î‡πÄ‡∏≠‡∏á)
            self.robot.move_to(300, 200, 50, 0)
            time.sleep(1)
            
            # 7. Release
            self.robot.suck(False)
            time.sleep(0.5)
            
            print("‚úì Grasp executed successfully!")
            return True
            
        except Exception as e:
            print(f"‚úó Grasp execution failed: {e}")
            return False
    
    def emergency_stop(self):
        """Emergency stop"""
        if self.robot:
            self.robot.close()
    
    def disconnect(self):
        """Disconnect robot"""
        if self.robot:
            self.robot.close()
            self.is_connected = False
            print("Robot disconnected")
```
