## ü§ñ ‡∏Å‡∏≤‡∏£‡∏ô‡∏≥ PCA ‡∏°‡∏≤‡πÉ‡∏ä‡πâ‡∏Å‡∏±‡∏ö Robotic Grasping

### ‡πÅ‡∏ô‡∏ß‡∏Ñ‡∏¥‡∏î‡∏´‡∏•‡∏±‡∏Å

> **"‡∏´‡∏¢‡∏¥‡∏ö‡∏ß‡∏±‡∏ï‡∏ñ‡∏∏‡∏à‡∏≤‡∏Å‡∏î‡πâ‡∏≤‡∏ô‡∏™‡∏±‡πâ‡∏ô ‡∏ï‡∏±‡πâ‡∏á‡∏â‡∏≤‡∏Å‡∏Å‡∏±‡∏ö‡∏î‡πâ‡∏≤‡∏ô‡∏¢‡∏≤‡∏ß"**

‡πÄ‡∏´‡∏°‡∏∑‡∏≠‡∏ô‡∏Å‡∏±‡∏ö‡∏ó‡∏µ‡πà‡∏°‡∏ô‡∏∏‡∏©‡∏¢‡πå‡πÉ‡∏ä‡πâ‡∏™‡∏±‡∏ç‡∏ä‡∏≤‡∏ï‡∏ç‡∏≤‡∏ì:
- ‡∏õ‡∏≤‡∏Å‡∏Å‡∏≤ ‚Üí ‡∏´‡∏¢‡∏¥‡∏ö‡∏Ç‡∏ß‡∏≤‡∏á (‡πÑ‡∏°‡πà‡πÉ‡∏ä‡πà‡∏´‡∏¢‡∏¥‡∏ö‡∏ï‡∏≤‡∏°‡∏¢‡∏≤‡∏ß)
- ‡∏Å‡∏•‡πà‡∏≠‡∏á‡∏¢‡∏≤‡∏ß ‚Üí ‡∏´‡∏¢‡∏¥‡∏ö‡∏î‡πâ‡∏≤‡∏ô‡πÅ‡∏Ñ‡∏ö

---

### Code ‡∏™‡∏≥‡∏´‡∏£‡∏±‡∏ö Grasp Selection

```python
import numpy as np
import cv2

def pca_grasp_analysis(contour):
    """
    ‡∏ß‡∏¥‡πÄ‡∏Ñ‡∏£‡∏≤‡∏∞‡∏´‡πå contour ‡∏î‡πâ‡∏ß‡∏¢ PCA ‡πÄ‡∏û‡∏∑‡πà‡∏≠‡∏´‡∏≤‡∏à‡∏∏‡∏î‡∏à‡∏±‡∏ö‡∏ó‡∏µ‡πà‡∏î‡∏µ‡∏ó‡∏µ‡πà‡∏™‡∏∏‡∏î
    """
    # Step 1: ‡πÅ‡∏õ‡∏•‡∏á contour ‡πÄ‡∏õ‡πá‡∏ô array ‡∏Ç‡∏≠‡∏á‡∏à‡∏∏‡∏î
    pts = contour.reshape(-1, 2).astype(np.float64)
    
    # Step 2: ‡∏´‡∏≤‡∏à‡∏∏‡∏î‡∏®‡∏π‡∏ô‡∏¢‡πå‡∏Å‡∏•‡∏≤‡∏á
    center = np.mean(pts, axis=0)
    
    # Step 3: ‡πÄ‡∏•‡∏∑‡πà‡∏≠‡∏ô‡∏à‡∏∏‡∏î‡πÉ‡∏´‡πâ‡∏®‡∏π‡∏ô‡∏¢‡πå‡∏Å‡∏•‡∏≤‡∏á‡∏≠‡∏¢‡∏π‡πà‡∏ó‡∏µ‡πà origin
    pts_centered = pts - center
    
    # Step 4: ‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì Covariance Matrix
    cov = np.cov(pts_centered.T)
    
    # Step 5: ‡∏´‡∏≤ Eigenvalues ‡πÅ‡∏•‡∏∞ Eigenvectors
    eigenvalues, eigenvectors = np.linalg.eig(cov)
    
    # Step 6: ‡πÄ‡∏£‡∏µ‡∏¢‡∏á‡∏•‡∏≥‡∏î‡∏±‡∏ö (‡∏°‡∏≤‡∏Å‡πÑ‡∏õ‡∏ô‡πâ‡∏≠‡∏¢)
    idx = np.argsort(eigenvalues)[::-1]
    
    major_axis = eigenvectors[:, idx[0]]  # ‡πÅ‡∏Å‡∏ô‡∏¢‡∏≤‡∏ß
    minor_axis = eigenvectors[:, idx[1]]  # ‡πÅ‡∏Å‡∏ô‡∏™‡∏±‡πâ‡∏ô
    
    # Step 7: ‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì‡∏°‡∏∏‡∏° grasp (‡∏ï‡∏±‡πâ‡∏á‡∏â‡∏≤‡∏Å‡∏Å‡∏±‡∏ö‡πÅ‡∏Å‡∏ô‡∏¢‡∏≤‡∏ß = ‡∏Ç‡∏ô‡∏≤‡∏ô‡∏Å‡∏±‡∏ö‡πÅ‡∏Å‡∏ô‡∏™‡∏±‡πâ‡∏ô)
    major_angle = np.degrees(np.arctan2(major_axis[1], major_axis[0]))
    grasp_angle = major_angle + 90  # ‡∏ï‡∏±‡πâ‡∏á‡∏â‡∏≤‡∏Å!
    
    # Step 8: ‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì‡∏Ñ‡∏ß‡∏≤‡∏°‡∏Å‡∏ß‡πâ‡∏≤‡∏á grasp (‡∏ï‡∏≤‡∏°‡πÅ‡∏Å‡∏ô‡∏™‡∏±‡πâ‡∏ô)
    projections = np.dot(pts_centered, minor_axis)
    grasp_width = np.max(projections) - np.min(projections)
    
    return {
        'center': center,
        'grasp_angle': grasp_angle,
        'grasp_width': grasp_width,
        'major_axis': major_axis,
        'minor_axis': minor_axis
    }
```

## üìê ‡∏™‡∏π‡∏ï‡∏£‡∏Ñ‡∏ì‡∏¥‡∏ï‡∏®‡∏≤‡∏™‡∏ï‡∏£‡πå

### Covariance Matrix (2D)

$$
\text{Cov} = \begin{bmatrix} 
\text{var}(X) & \text{cov}(X,Y) \\
\text{cov}(X,Y) & \text{var}(Y)
\end{bmatrix}
$$

‡πÇ‡∏î‡∏¢‡∏ó‡∏µ‡πà:
- $\text{var}(X) = \frac{1}{n} \sum_{i=1}^{n} (x_i - \bar{x})^2$
- $\text{cov}(X,Y) = \frac{1}{n} \sum_{i=1}^{n} (x_i - \bar{x})(y_i - \bar{y})$

### Eigenvalue Equation

$$
\text{Cov} \cdot \vec{v} = \lambda \cdot \vec{v}
$$

‡πÇ‡∏î‡∏¢‡∏ó‡∏µ‡πà:
- $\vec{v}$ = Eigenvector (‡∏ó‡∏¥‡∏®‡∏ó‡∏≤‡∏á‡∏Ç‡∏≠‡∏á‡πÅ‡∏Å‡∏ô)
- $\lambda$ = Eigenvalue (‡∏Ñ‡∏ß‡∏≤‡∏°‡∏™‡∏≥‡∏Ñ‡∏±‡∏ç‡∏Ç‡∏≠‡∏á‡πÅ‡∏Å‡∏ô)

    """
    ‡∏ß‡∏≤‡∏î‡πÅ‡∏Å‡∏ô PCA ‡∏ö‡∏ô frame
    """
    cx, cy = int(center[0]), int(center[1])
    
    # ‡∏ß‡∏≤‡∏î‡πÅ‡∏Å‡∏ô‡∏¢‡∏≤‡∏ß (‡∏™‡∏µ‡πÅ‡∏î‡∏á)

                 int(cy + major_axis[1] * scale))

    cv2.putText(frame, "Major", end_major, 
                cv2.FONT_HERSHEY_SIMPLEX, 0.5, (0, 0, 255), 1)

    end_minor = (int(cx + minor_axis[0] * scale), 
                 int(cy + minor_axis[1] * scale))
    cv2.line(frame, (cx, cy), end_minor, (0, 255, 0), 2)
    cv2.putText(frame, "Grasp", end_minor, 
                cv2.FONT_HERSHEY_SIMPLEX, 0.5, (0, 255, 0), 1)
    
    # ‡∏ß‡∏≤‡∏î‡∏à‡∏∏‡∏î‡∏®‡∏π‡∏ô‡∏¢‡πå‡∏Å‡∏•‡∏≤‡∏á
    cv2.circle(frame, (cx, cy), 5, (255, 0, 0), -1)
    
    return frame
